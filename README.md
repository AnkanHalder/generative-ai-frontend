## Getting Started

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```sh
   cd <project-directory>
   ```
3. Install the dependencies:
   ```sh
   npm ci
   ```

### Development

To start the development server, run:

```sh
npm run dev

```
